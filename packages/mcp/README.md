# @inspect/mcp

Model Context Protocol (MCP) server for inspect. Exposes browser automation capabilities as MCP tools for AI agents.

## Usage

```ts
// Run the MCP server
import { McpServer } from "@inspect/mcp/server.js";
```

## Key Exports

- `McpServer` — MCP server with browser tools
- `inspect-mcp` — CLI binary for running the MCP server
