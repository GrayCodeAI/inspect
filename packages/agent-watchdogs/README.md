# @inspect/agent-watchdogs

Monitoring and supervision for autonomous agents. Detects anomalies, infinite loops, and off-course behavior during test execution.

## Usage

```ts
import { Watchdog } from "@inspect/agent-watchdogs/watchdog.js";
```

## Key Exports

- `Watchdog` — monitors agent health and behavior patterns
- `AnomalyDetector` — identifies suspicious agent activity
