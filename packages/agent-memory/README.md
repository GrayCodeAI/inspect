# @inspect/agent-memory

Persistent memory layer for AI agents. Stores conversation history, learned patterns, and contextual state across test sessions.

## Usage

```ts
import { AgentMemory } from "@inspect/agent-memory/agent-memory.js";
```

## Key Exports

- `AgentMemory` — manages agent state persistence and retrieval
- `MemoryStore` — abstraction for storing and querying agent memories
