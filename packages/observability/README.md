# @inspect/observability

Structured logging, tracing, and metrics for inspect. Provides Effect-native observability primitives used across all packages.

## Usage

```ts
import { Trace } from "@inspect/observability/trace.js";
```

## Key Exports

- `Trace` — distributed tracing utilities
- `MetricsCollector` — performance and operational metrics
- `StructuredLogger` — Effect-based structured logging
