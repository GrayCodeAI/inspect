# @inspect/desktop

Desktop automation and computer use capabilities. Enables inspect to interact with native desktop applications via screen capture and input simulation.

## Usage

```ts
import { DesktopController } from "@inspect/desktop/desktop-controller.js";
```

## Key Exports

- `DesktopController` — manages desktop automation
- `ScreenCapture` — screenshot capture utility
- `InputSimulator` — keyboard and mouse simulation
