# @inspect/lighthouse-quality

Lighthouse integration for quality auditing. Runs performance, accessibility, SEO, and best-practice audits during inspect test runs.

## Usage

```ts
import { LighthouseAudit } from "@inspect/lighthouse-quality/lighthouse-audit.js";
```

## Key Exports

- `LighthouseAudit` — runs Lighthouse audits against pages
- `LighthouseReport` — schema for Lighthouse results
