# @inspect/multi-agent

Multi-agent test orchestration. Coordinates multiple AI agents working in parallel to test different aspects of an application simultaneously.

## Usage

```ts
import { MultiAgentOrchestrator } from "@inspect/multi-agent/multi-agent-orchestrator.js";
```

## Key Exports

- `MultiAgentOrchestrator` — coordinates parallel agent execution
- `AgentRole` — defines agent responsibilities in multi-agent scenarios
