# @inspect/browser

Playwright-based browser automation with MCP protocol support and rrweb session recording. Manages browser lifecycle, page interactions, and CDP connections.

## Usage

```ts
import { BrowserManager } from "@inspect/browser/browser-manager.js";
```

## Key Exports

- `BrowserManager` — browser lifecycle and page management
- `CdpClient` — Chrome DevTools Protocol client
- `SessionRecorder` — rrweb-based session recording
