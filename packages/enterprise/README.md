# @inspect/enterprise

Enterprise features for inspect. Provides SSO integration, team management, audit logging, and organization-level configuration.

## Usage

```ts
import { EnterpriseConfig } from "@inspect/enterprise/enterprise-config.js";
```

## Key Exports

- `EnterpriseConfig` — enterprise-level configuration management
- `AuditLogger` — compliance audit logging
- `TeamManager` — organization and team management
