# @inspect/session-recording

Session recording and replay for browser tests using rrweb. Captures full DOM-level recordings of test sessions for visual debugging and review.

## Usage

```ts
import { SessionRecorder } from "@inspect/session-recording/session-recorder.js";
```

## Key Exports

- `SessionRecorder` — rrweb-based session capture
- `SessionPlayer` — playback recorded sessions
- `Recording` — schema for recorded session data
