# @inspect/sdk

Public SDK for integrating inspect into applications. Provides programmatic access to browser testing, agent orchestration, and quality analysis.

## Usage

```ts
import { InspectSDK } from "@inspect/sdk/inspect-sdk.js";
```

## Key Exports

- `InspectSDK` — main SDK entry point
- `TestRunner` — programmatic test execution
- `QualityAnalyzer` — quality analysis API
