# @inspect/agent

AI agent integration layer. Connects coding agents (Claude Code, Codex, Cursor, Copilot, Gemini) to the inspect testing infrastructure via the Agent Client Protocol.

## Usage

```ts
import { Agent } from "@inspect/agent/agent.js";
```

## Key Exports

- `Agent` — unified interface for coding agent communication
- `AgentSession` — manages a single agent conversation session
