# @inspect/visual-builder

Visual test builder for creating tests via a UI. Enables drag-and-drop test creation with LLM-assisted step generation.

## Usage

```ts
import { VisualBuilder } from "@inspect/visual-builder/visual-builder.js";
```

## Key Exports

- `VisualBuilder` — UI-based test creation interface
- `TestStepBuilder` — visual step composition
