# @inspect/a11y

Accessibility auditing for browser-based tests. Integrates with the orchestrator to run WCAG compliance checks during test execution.

## Usage

```ts
import { A11yChecker } from "@inspect/a11y/a11y-checker.js";
```

## Key Exports

- `A11yChecker` — runs accessibility audits against live pages
- `A11yViolation` — schema for accessibility findings
