# @inspect/yaml

YAML parsing, generation, and test execution for inspect. Handles YAML-based test configuration and workflow definitions.

## Usage

```ts
import { YamlParser } from "@inspect/yaml/parser.js";
import { YamlGenerator } from "@inspect/yaml/generator.js";
```

## Key Exports

- `YamlParser` — parses YAML test definitions
- `YamlGenerator` — generates YAML from test configurations
- `YamlRunner` — executes tests defined in YAML
