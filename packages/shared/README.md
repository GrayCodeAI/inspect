# @inspect/shared

Shared domain models, types, schemas, and utilities used across all inspect packages. The foundational layer that every other package depends on.

## Usage

```ts
import { TestPlan } from "@inspect/shared/test-plan.js";
import { ExecutionEvent } from "@inspect/shared/execution-event.js";
```

## Key Exports

- `TestPlan` — test plan domain model
- `ExecutionEvent` — test execution event schema
- `Error` classes — shared error types across packages
